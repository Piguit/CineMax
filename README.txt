Istruzioni per l'installazione e l'avvio

Prima di tutto, è necessario estrarre i file dall'archivio compresso:

1. Fare clic destro sul file .zip scaricato.
2. Selezionare "Estrai tutto..." (su Windows) o fare doppio clic sul file (su macOS o Linux).
3. Scegliere una cartella di destinazione a tua scelta e conferma l'estrazione; evitare percorsi protetti, come ad esempio Program Files.

Una volta estratta la cartella, seguire questi passaggi:

1. Aprire il terminale (Prompt dei comandi o PowerShell su Windows, Terminale su macOS o Linux).
2. Spostarsi all'interno della cartella bin che si trova nella directory appena estratta. Ad esempio:
	macOS -> cd /Utenti/nomeutente/Desktop/CineMax-main/bin
	Windows -> cd C:\Utenti\nomeutente\Desktop\CineMax-main\bin
3. Digitare e inviare il comando: java -jar CineMax.jar
